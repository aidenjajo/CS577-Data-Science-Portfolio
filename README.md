## üìä Projects Overview

### 1. [Data Preprocessing Pipeline](./01-Data-Preprocessing)
Implements a complete data preprocessing workflow including missing value imputation, label encoding, train-test split, and feature scaling.
- **Technologies**: Python, pandas, scikit-learn
- **Key Skills**: Data cleaning, feature engineering, standardization

### 2. [Regression Models Comparison](./02-Regression-Models-Comparison)
Compares five regression algorithms (Random Forest, Decision Tree, Linear, Polynomial, SVR) on power plant data.
- **Technologies**: Python, scikit-learn, NumPy
- **Key Skills**: Regression analysis, model comparison, MSE/R¬≤ evaluation

### 3. [Regression Analysis](./03-Regression-Analysis)
Demonstrates simple linear, multiple, and polynomial regression with comprehensive visualizations.
- **Technologies**: Python, scikit-learn, matplotlib
- **Key Skills**: Linear modeling, polynomial features, categorical encoding

### 4. [Classification Models Comparison](./04-Classification-Models)
Implements and compares 7 classification algorithms with GridSearchCV optimization and extensive visualizations.
- **Technologies**: Python, scikit-learn, matplotlib, seaborn
- **Key Skills**: Classification, hyperparameter tuning, decision boundaries, confusion matrices

### 5. [Titanic Survival Prediction](./05-Titanic-Survival-Prediction)
Predicts passenger survival on the Titanic using 6 classification models with hyperparameter optimization.
- **Technologies**: Python, scikit-learn
- **Key Skills**: Binary classification, feature engineering, GridSearchCV

### 6. [Airline Passenger Satisfaction](./06-Airline-Satisfaction)
Analyzes airline satisfaction data using Random Forest and XGBoost with automated PDF reporting.
- **Technologies**: Python, XGBoost, scikit-learn, fpdf
- **Key Skills**: Ensemble methods, automated reporting, stratified sampling

### 7. [Imbalanced Data Handling with SMOTE](./07-Imbalanced-Data-SMOTE)
Addresses class imbalance in credit card fraud detection using SMOTE technique.
- **Technologies**: Python, imbalanced-learn, scikit-learn
- **Key Skills**: SMOTE, handling imbalanced datasets, fraud detection

### 8. [ANN Customer Churn Prediction](./08-ANN-Customer-Churn)
Builds an Artificial Neural Network to predict customer churn in banking using TensorFlow/Keras.
- **Technologies**: Python, TensorFlow, Keras
- **Key Skills**: Deep learning, neural networks, binary classification, early stopping

### 9. [CNN for MNIST Digit Classification](./09-CNN-MNIST)
Implements a Convolutional Neural Network for handwritten digit recognition achieving ~94% accuracy.
- **Technologies**: Python, TensorFlow, Keras
- **Key Skills**: Computer vision, CNNs, image classification

### 10. [CNN for CIFAR-10 Image Classification](./10-CNN-CIFAR10)
Advanced CNN with batch normalization and dropout for multi-class image classification achieving ~81% accuracy.
- **Technologies**: Python, TensorFlow, Keras, scikit-learn
- **Key Skills**: Deep CNNs, batch normalization, dropout regularization, multi-class classification
